# 信贷风险评估可视化分析预测系统

#### 介绍
Hadoop机器学习信贷风险评估可视化分析预测系统基于Java 语言，融合了Hadoop，spark，hive大数据处理技术、Spring Boot 框架以及 MySQL 数据库，结合机器学习算法随机森林回归，数据分析数据清洗等功能。
